## Descripción [Este proyecto esta relacionado con la salud publica a nivel mundial, una estructura muy completa para realizar un analisis profundo y exaustivo, cubre desde determinantes economicos y sociales hasta resultados clinicos y factores de riesgos conductuales] 

## Dataset [Esta estructura de datos contiene 159 paises de todos los continentes a su vez dividos por regiones, tiene variables socioeconomicas(nivel_ingresos, poblacion_urbana_pct), tambien tiene variables de tipo inversion o recursos tales como (gastos_salud_usd, medicos_por_1000, camas_hospital_por_1000), contiene factores de riesgo para predecir cargas futuras (obesidad_pct, diabetes_pct, tabaquismo_pct), a su vez tiene unos items de prevencion como son las (vacunacion_pct), todo esto para ayudar a resolver y obtener resultados eficientes en variables como (esperanza_vida, mortalidad_infantil)]
